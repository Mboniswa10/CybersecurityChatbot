## Project Overview

The Cybersecurity Awareness Assistant is a C# WPF desktop application designed to educate users about cybersecurity concepts and online safety practices.

The chatbot provides cybersecurity guidance, stores user information, manages tasks, tracks user activity, performs sentiment detection, and includes an interactive cybersecurity quiz.

The project demonstrates the use of Object-Oriented Programming (OOP), data structures, GUI development, Natural Language Processing (NLP) simulation, task management, memory storage, and activity logging.

---

## Features

### User Registration

* Captures user's name.
* Captures student number.
* Remembers user information throughout the session.

### Cybersecurity Knowledge Base

The chatbot can answer questions about:

* Password Security
* Phishing
* Malware
* VPN
* Privacy
* Firewalls
* Encryption
* Viruses
* Ransomware
* Spyware
* Data Breaches
* Social Media Safety
* Cyberbullying
* Identity Theft
* Wi-Fi Security
* Two-Factor Authentication
* Hackers
* Cybersecurity Awareness

### Sentiment Detection

The chatbot detects emotions such as:

* Happy
* Sad
* Worried
* Angry
* Frustrated
* Curious
* Tired

and responds appropriately.

### User Memory

The chatbot remembers:

* User Name
* Student Number
* Favourite Cybersecurity Topic

### Interactive Cybersecurity Quiz

Features:

* True/False questions
* Multiple questions
* Automatic scoring
* Immediate feedback
* Final score calculation
* Performance evaluation

### Task Management

Users can:

* Add Tasks
* View Tasks
* Complete Tasks
* Delete Tasks
* Set Reminders

### Activity Logging

The application records:

* User interactions
* Quiz activity
* Task activity
* System actions

### Natural Language Processing (NLP) Simulation

Recognizes commands such as:

* start quiz
* add task
* show tasks
* show activity log
* remind me
* create task

### Graphical User Interface (GUI)

Built using Windows Presentation Foundation (WPF).

Includes:

* Chat Interface
* Task Panel
* Activity Log Panel
* Send Button
* Application Branding
